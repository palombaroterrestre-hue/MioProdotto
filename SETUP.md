# MioProdotto - Frontend Setup

## Quick Start

### 1. Crea tabella watchlist su Supabase

Vai su **Supabase SQL Editor** ed esegui:

```sql
-- Crea tabella watchlist
CREATE TABLE IF NOT EXISTS watchlist (
  id BIGINT GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
  nome_prodotto TEXT NOT NULL,
  utente_id TEXT NOT NULL,
  created_at TIMESTAMP WITH TIME ZONE DEFAULT NOW()
);

-- Enable RLS
ALTER TABLE watchlist ENABLE ROW LEVEL SECURITY;

-- Policy per insert
CREATE POLICY "Allow insert" ON watchlist FOR INSERT WITH CHECK (true);
-- Policy per select
CREATE POLICY "Allow select" ON watchlist FOR SELECT USING (true);
```

### 2. Installa dipendenze

```bash
cd frontend
npm install
```

### 3. Avvia dev

```bash
npm run dev
```

### 4. Build produzione

```bash
npm run build
```
